# TechEdu E-Library

Digital learning platform for Secondary and Technical students.

## Features
- Student login / registration
- Teacher dashboard
- Downloadable lesson PDFs
- Objective and theory questions
- Practical exercises

## Subjects
Physics, Basic Electricity, Solar Installation, Electric Current

## Instructions
Open `index.html` to start. Students register/login to access lessons.
Teachers login with admin code (1234) to manage lessons.
